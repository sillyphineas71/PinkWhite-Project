# Backend Specs

Khu vực spec cho backend Social Matchmaking Platform.

## Cấu trúc

- `features/`: mô tả feature theo nghiệp vụ và acceptance criteria.
- `modules/`: mô tả thiết kế kỹ thuật theo NestJS module.
- `global/`: quy ước chung cho toàn backend.

## Quy tắc

- Viết feature spec trước khi implement.
- Nếu feature làm thay đổi API/DB/realtime, cập nhật module spec tương ứng.
- Frontend spec chưa nằm trong phạm vi hiện tại.
