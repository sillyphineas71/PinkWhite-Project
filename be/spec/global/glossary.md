# Glossary

- Account: danh tính đăng nhập của user.
- Profile: hồ sơ hiển thị trong discovery.
- Preference: tiêu chí tìm kiếm của user.
- Discovery: luồng gợi ý profile để swipe.
- Swipe: hành động like/pass/superlike nếu sau này có.
- Match: quan hệ được tạo khi hai user cùng like nhau.
- Active match: match còn hiệu lực, chưa unmatch/block.
- Report: báo cáo user hoặc nội dung vi phạm.
- Block: chặn tương tác giữa hai user.
